This utility was written in C# using Visual Studio 2012 as an IDE.

Usage:

1) place the original simulation xml file into the folder named original
2) place as many simulation xml files wiht altered configurations into the folder called changes
3) run the file SimXmlToCSV.exe
4) follow the onscreen prompts.
5) all output files will be created in the same directory as SimXmlToCSV.exe
